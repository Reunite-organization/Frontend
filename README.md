# Reunite
Reunite is a platform for missing and wanting people to connect. It allows users to create profiles, share their stories, and search for others who may be missing or looking for someone. The platform also provides resources and support for those affected by missing persons cases.

## Features
- User profiles: Create and manage your profile with personal information, photos, and stories.
- Search functionality: Search for missing or wanting individuals based on various criteria such as location, age, and physical characteristics.
- Community support: Connect with others who have similar experiences and share resources and support.
- Notifications: Receive updates and notifications about new profiles and relevant information.
- Privacy settings: Control who can see your profile and information.
## Technologies Used
- Frontend: React, Redux, Material-UI
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: JWT (JSON Web Tokens)
- Deployment:Render
## Installation
1. Clone the repository: `git clone https://github.com/reunite/reunite.git`
2. Navigate to the client directory: `cd client`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
